Суть проекта

Основная цель приложения — продемонстрировать работу с многопоточностью (threading), обработку событий остановки потоков и логирование действий. Каждая задача работает в отдельном "фоновом" потоке, не блокируя основной интерфейс управления.

Основные возможности:
• Динамическое управление: Добавление и удаление задач "на лету" без остановки программы.
• Многопоточность: Каждая задача выполняется в своем потоке с заданным интервалом.
• Логирование: Все действия (запуск, выполнение вычислений, ошибки, остановка) записываются в файл tasks.log и выводятся в консоль.
• Уникальные ID: Каждой задаче присваивается короткий уникальный идентификатор (UUID) для удобного управления.
• Безопасность: Проверка на слишком большие значения (предотвращение OverflowError) и корректное завершение потоков при выходе.

Как пользоваться

1. Требования
Для запуска проекта необходим только установленный Python 3.6 или выше. Дополнительные библиотеки (pip) не требуются, так как используются стандартные модули.

2. Запуск
Склонируйте проект или скопируйте код в файл (например, main.py). Запустите его командой:
python main.py

3. Интерактивное меню
После запуска вы увидите меню управления:

4.  Добавить новую задачу:
• Введите имя (например, "Расчет мощности").
• Укажите интервал в секундах (как часто выполнять).
• Укажите основание (число) и степень для вычисления.
• *Задача сразу начнет выполняться в фоне.*
2.  Удалить задачу:
• Введите короткий ID задачи (его можно узнать в списке задач). Поток будет остановлен и задача удалена из памяти.
3.  Список активных задач:
• Выводит таблицу со всеми созданными задачами, их параметрами, ID и текущим статусом.
4.  Выход:
• Корректно останавливает все запущенные потоки и закрывает программу.
